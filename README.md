# robot_learning

To run ADB on Bluestacks go to **Settings** &rarr; **Advanced** &rarr; **Android Debug Bridge (ADB)**. Save changes (in the bottom right) then restart Blue stacks

## Required Installation
`pip install pure-python-adb`

Then do this
### For mac:
`brew install android-platform-tools`

### OR just run this to get everything you need all at once (supposedly):
`pip install -r requirements.txt`
